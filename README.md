# Kanban Board App 📋

A task management application built with JavaScript that helps users organize tasks into different categories.

## Features

- Create tasks
- Move tasks between columns
- Edit tasks
- Delete tasks
- Organize tasks by status
- Responsive design

## Technologies

- HTML
- CSS
- JavaScript

## What I learned

- Working with JavaScript objects and arrays
- Managing application state
- Handling user interactions
- DOM manipulation
- Building interactive web applications
